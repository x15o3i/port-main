DocuVerify

Web Application  |  Document Authentication Platform

DocuVerify is a secure full-stack tool built to detect document forgery. It processes suspect documents against master templates using Structural Similarity (SSIM) algorithms and geometric anchor point analysis. The platform features bulk batch evaluations, secure user sessions, and a comprehensive audit log to track all verification histories.

Tools: Python, Flask, OpenCV, scikit-image, SQLite, HTML/CSS/JS
